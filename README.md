## ExtraSyntax.vim
This appends some keyword to syntax of 'Todo'.
For example, c syntax NOT highlighting 'NOTE' keyword like 'FIXME', 'XXX', 'TODO'.
You can append 'NOTE' using this plugin.

## installation
```vim
NeoBundle 'mopp/ExtraSyntax.vim'
```
